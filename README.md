# Django Ecommerce

This is a simple e-commerce website built with Django.

Some of the features:
  - Ecommerce system with cart and checkout functionality.
  - Nice Frontend UI with templates.
  - Email sending configuration with SMTP and Celery.
  - Restful APIs
  - Oauth
  - Dockerized
